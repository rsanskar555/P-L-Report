P&L Report – AtliQ Hardware
📌 Overview
A comprehensive Profit & Loss report for AtliQ Hardware for FY 2022, built entirely in Excel. The report provides granular insights into revenue streams and expense categories, powered by advanced Excel features and automation.
🛠️ Tech Stack

Microsoft Excel – Core reporting environment
Pivot Tables – Dynamic summarization of financial data across dimensions
Power Query – Automated ETL pipeline for data cleaning and transformation
DAX – Calculation models for accuracy and flexibility

✨ Key Highlights

Generated a full P&L breakdown covering gross price, pre/post-invoice deductions, net sales, gross margin, COGS, and net profit for FY 2022.
Automated repetitive data cleaning workflows using Power Query, saving approximately 4 hours of manual work per reporting cycle.
Built DAX-based calculation models that improved report accuracy by 30%, enabling more nuanced and flexible financial insights.
Designed the report structure to allow drill-down by product category, market, and customer channel (Retailers, Direct, Distributors).

📈 Business Impact

Empowered finance stakeholders with a reliable, repeatable reporting process.
Reduced human error from manual data manipulation.
Provided a scalable foundation for month-over-month and year-over-year P&L comparisons.
